    Saving the tape

    In Visual Turing you can save certain configurations of the tape that can be loaded later. There are three buttons on the tape toolbar that help you to save the tape with the name typed in the edit ofthe combo box, to delete the tape with the name in the same edit and to create a new tape. The tapes can be loaded when the combo box drops down. They are saved in the project file.
